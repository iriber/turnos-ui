En la agenda vas a ver todos los turnos asignados al médico.

En cada uno de estos turnos asignados vas a ver la opción para ponerlo en sala:
{asignado_label} {ponerEnSala_msg}


Clickeando sobre esta opción pueden ocurrir dos casos:
  1. El paciente ya se encuentra registrado asi que únicamente clickeando en la opción ya queda el paciente en sala
  2. El paciente aún no fue registrado. En este caso, el sistema nos llevará a registrar el paciente para luego sí dejarlo en sala
