En la agenda vas a ver todos los turnos disponibles que tiene el médico para el día seleccionado

Cada renglón tiene una hora. Si el renglón está vacío para la hora que buscás, entonces yo podés asignar un paciente al turno clickeando sobre el renglón.

Se te abrirá una página para que ingreses los datos del turno: Paciente, Obra social, Nro de Afiliado, Estado e Importe.

Es importante que elijas el Estado correcto para el turno.
